# 아카호시

아카호시는 조선일보의 [헬스조선](https://health.chosun.com/), 그리고 [일본어 기사](https://www.chosun.com/national/nie/japanese/)를
수집하여 워드프레스의 포스트로 저장하는 플러그인입니다.

## 사용법

1. WordPress 관리자 페이지에 접속합니다.
2. '플러그인' 메뉴를 선택하고 '설치된 플러그인' 탭으로 이동합니다.
3. '아카호시' 플러그인을 활성화합니다.
4. '설정' 메뉴를 선택하고 '아카호시 설정'으로 이동합니다.
5. '일본어 스크랩'과 '건강 스크랩' 섹션의 키워드 필드에 각각 알맞은 키워드를 입력합니다.
   '활성화'에 체크해야 스크랩이 동작합니다. 
6. '저장' 버튼을 클릭하여 설정을 저장합니다.

## 설정하기

일본어 스크랩에는 키워드 '\[입에 착착 붙는 일본어]',
건강 스크랩에는 각자 알맞는 키워드를 입력합니다. 기사 제목에서 해당 키워드를 포함하면 스크랩됩니다.

기사 스크랩은 매 시간마다 동작합니다.
